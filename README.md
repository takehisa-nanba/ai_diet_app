# FitTracker (AI ダイエットコーチ)

無理なく健康的に目標体重に近づけるための、AI搭載型ダイエット補助・記録アプリです。

## 概要

日々の体重を記録し、ダッシュボードで進捗を視覚的に確認することでモチベーションを維持できます。
最大の特徴として、**ローカル環境でファインチューニングされた独自のAIコーチ**がユーザーの目標や現在の体重（BMI等）に合わせたダイエットプランの提案と、日々の相談（チャットアシスト）を行ってくれます。

## 主な機能

1. **ダッシュボード**
   - 直近2週間の体重推移グラフ
   - 目標体重までのプログレスバー（達成率、残りkg数）
   - 日々の記録ステータス確認
2. **体重記録**
   - 毎日の体重の手軽な入力インターフェース
   - 直近30件の推移グラフ表示と、過去の記録一覧の管理（削除機能）
3. **AIトレーナー（プラン＆チャット統合）**
   - 独自のAIによる「無理なく健康的なダイエットプラン」の提案
   - 悩みや食事に関するAIコーチとのチャット相談
4. **プロフィール設定**
   - 身長・体重からBMIの自動計算
   - 目標体重、目標達成期限、日々の活動レベルの詳細設定

## 技術スタック

### フロントエンド (App)
- **React Native (Expo)**: iOS / Android / Web のクロスプラットフォーム対応
- **React Navigation**: 下部タブメニュー等の画面遷移
- **React Native Chart Kit**: 体重推移などのリッチなグラフ描画
- **AsyncStorage**: ユーザーデータや履歴のローカル保存

### バックエンド (Backend)
- **FastAPI (Python)**: フロントエンドからのチャットリクエストを処理する高速なAPIサーバー
- **Hugging Face Transformers / PyTorch**: AIモデルの読み込みと推論処理
- **AIモデル**: `cyberagent/open-calm-small` をベースに独自のダイエット知識データ（`diet_dataset.jsonl`）でCPUファインチューニングを実施した自作AIモデル（ローカル動作）

## 起動方法

### バックエンドの起動
```bash
cd backend
# 仮想環境のアクティベート (Windows)
.\venv\Scripts\Activate.ps1
# サーバーの起動
uvicorn main:app --reload
```

### フロントエンドの起動
```bash
cd app
# パッケージのインストール
npm install
# Expoサーバーの起動
npm run start
# もしくはWebブラウザで開く場合
npm run web
```
